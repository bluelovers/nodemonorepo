# sh-or-cmd

Returns `'sh'` in UNIX/Linux and `'cmd'` in Windows.

## Requirements

* Node.js ≥ 8.9.0

## Usage

```javascript
import shell from 'sh-or-cmd'
console.log(shell)
```

It prints either `'sh'` or `'cmd'`
