# smart-publish

Like `npm publish`, but also tag prepreleases and knows when it is public

## Requirements

* Node.js ≥ 8.9.0

## Usage

### Basic

```sh
cd <package-directory>
smart-publish
```

### Advanced

```sh
export NPM=<npm-like-command>
export CWD=<package-directory>
smart-publish
```
